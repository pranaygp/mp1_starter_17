CSS Peek

CSS Peek is an extension for the famous text editor, Visual Studio Code. It gives you "Go To Definition", "PeeK" (hovering on an identifier to see its definition) and "Inline editing" functionality of CSS classNames and IDs to be able to peek and edit the actual CSS content from within a HTML file when "peek" - ing. It currently has over 40K installs and has some great reviews from the open source community.

ZEIT Deployments

ZEIT deployments is a little app in React that simply uses the now API from ZEIT to display a listing of all deployments made to the ZEIT servers using now. It is an awesome way for you to share a file with yourself when you upload something from your personal laptop using now, and then jump over to a public computer, go to the URL where you deployed ZEIT Deployments and download the file you ust uploaded.

Relay

Relay is an open source declerative data fetching library for Javascript managed by Facebook. It serves as a client library to help cosume data from a server that implements the GraphQL spec. Amongst other things, Relay provides abstractions for things like Request/Response caching, pagination and data refetching, streaming requests over polling/HTTP/2 Server push, etc.
